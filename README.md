# datacrash.github.io

Landing page raiz da presença profissional DataCrash.

## Objetivo

Este repositório hospeda o ponto de entrada principal em `datacrash.github.io`, funcionando como camada de navegação rápida para:

- Professional Hub
- Currículo PT-BR
- Resume EN
- GitHub profile README
- LinkedIn

## Stack

- React 19
- TypeScript 6
- Vite 8
- ESLint 10
- GitHub Pages

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Deploy

O deploy é feito via GitHub Actions para GitHub Pages.

## Relação com outros repositórios

- `DataCrash/datacrash`: README especial do perfil GitHub
- `DataCrash/datacrash.github.io`: landing root do domínio público
- `DataCrash/professional-hub`: hub detalhado com dashboard, CVs e narrativa profissional
